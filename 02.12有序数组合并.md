# 题目描述
有两个从小到大排序以后的数组A和B，其中A的末端有足够的缓冲空容纳B。请编写一个方法，将B合并入A并排序。

给定两个有序int数组A和B，A中的缓冲空用0填充，同时给定A和B的真实大小int n和int m，请返回合并后的数组。

# 代码
```cpp
class Merge {
public:
    int* mergeAB(int* A, int* B, int n, int m) {
        // write code here
        int pA=n-1;
        int pB=m-1;
        int pMerge=m+n-1;
        while(pA>=0 || pB>=0){
            if(A[pA]<=B[pB]){
                A[pMerge]=B[pB];
                pB--;
                pMerge--;
            }
            else{
                A[pMerge]=A[pA];
                pA--;
                pMerge--;
            }
        }
        while(pB>=0){
            A[pMerge--]=B[pB--];
        }
        return A;
    }
};
```
